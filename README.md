# La Main d’Or — One-page (GitHub Pages)

## Déploiement (GitHub Pages)
1. Crée un repo GitHub (ex: `la-main-dor`).
2. Upload le contenu du dossier `site-final/` à la racine du repo.
3. GitHub → **Settings** → **Pages** → Source: `Deploy from a branch` → Branch: `main` / `(root)` → Save.
4. Ton site sera accessible à une URL du type : `https://<ton-compte>.github.io/<repo>/`

## À modifier après mise en ligne
- Dans `index.html` :
  - Remplacer **[URL_DU_SITE]** (canonical + og:url + og:image + schema)
- Dans `robots.txt` et `sitemap.xml` :
  - Remplacer **[URL_DU_SITE]** par l’URL GitHub Pages finale.
- Option : remplacer le lien “Me trouver sur Google” et “Avis Google” si tu as l’URL exacte de ta fiche.

## Images
